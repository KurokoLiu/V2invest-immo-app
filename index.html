Sous-paramètres (Excel → web)

TF/TEOM et Loyer/chambre peuvent se remplir automatiquement via les bases par Code postal. La surface de référence par chambre est ajustable.

Utilisé pour calculer un loyer/chambre auto = loyer_m² × surfaceRef.
Notes de calcul
  • Coupon investisseur (phase 1) = intérêts seuls sur l’Avance SASU (mensualisé).
  • Coupon investisseur (phase 2) = mensualité SASU (amortissement + intérêts).
  • Mensualité banque / SASU = PMT(i, n, PV) avec i = taux/12, n = durée*12.
  • Charges mensuelles = Charges fixes + TF + TEOM + Provision%×Loyers + Gestion%×Loyers.
  • Revenus bruts = Nb chambres × Loyer/chambre (× Occupation).
  • Cash-flow net après impôt = (Revenus − Charges − mensualités) × (1 − Timpôt).
